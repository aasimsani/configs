# Custom Mappings

- ## Vim
    - g + hjkl to navigate between vim panes
    - <Space> opens NERDTree
- ## Tmux
  - Ctrl + B to Ctrl + A
  - Ctrl + A then - makes new horizontal pane
  - Ctrl + A then | makes new vertical pane
  - Ctrl + hjkl lets you navigate tmux panes
